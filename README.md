## ⭐️About Project

**2023年电子设计大赛D题，对任意调制信号的判断和测量。** 
<div>
 
***思路*** 通过Openmv采集数据，将识别后的结果传给STM32主控，将数据显示并驱动小车移动。
<div>

***主控STM32f407*** 通过PWM控制电机转动，通过Uart接收OpenMV数据并将数据通过串口屏显示。
<div>

***OpenMV：*** 用Edge implus 在线训练模型，对图片进行识别。。
<div>
